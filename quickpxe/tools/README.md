# BIOS Tools

Place the vendor BIOS tools in this folder:

## Dell
- `dell/cctk.exe` - Dell Command Configure (CCTK)
- Download from: https://www.dell.com/support/kbdoc/en-us/000178000/dell-command-configure

## HP
- `hp/BiosConfigUtility64.exe` - HP BIOS Configuration Utility
- `hp/BiosConfigUtility.exe` - 32-bit version
- Download from: https://ftp.hp.com/pub/caps-softpaq/cmit/HP_BCU.html

## Lenovo
- `lenovo/ThinkBiosConfig.hta` - Think BIOS Config Tool
- Download from: https://support.lenovo.com/solutions/ht100612

## Folder Structure

```
tools/
├── dell/
│   └── cctk.exe
├── hp/
│   ├── BiosConfigUtility64.exe
│   └── BiosConfigUtility.exe
└── lenovo/
    └── ThinkBiosConfig.hta
```
